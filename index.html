Plot Display

These controls only change the visual presentation of the selected data.

Circular Axis Controls how periodic angles are displayed without changing the underlying values.
Smoothing σ Gaussian smoothing width in histogram-bin units. Larger values flatten peaks.
Probability / Density Probability shows smoothed per-bin mass. Density additionally normalizes by bin width, including angular-bin width for circular variables.
Trace Style Switch between the current filled traces and line-only curves without changing the selected data.
Histogram Detail Controls the raw histogram resolution before smoothing. Standard uses 64 bins for linear parameters and 72 for circular parameters; Fine doubles this to 128 and 144. The selected bins are Gaussian-smoothed using the current Smoothing σ. Under Probability, the smoothed bins are normalized to sum to 1; under Density, they are additionally divided by bin width, including angular bin width for circular variables.

2D Joint Analysis

Select a second parameter to generate a joint distribution plot. Same-level pairing remains available, and pair-level parameters can also be expanded onto residue-level measurements for base-pair versus local torsion/pucker analysis.

Join Mode Same-level matches rows by PDB entry and identical site label. Pair -> Residue expands each pair observation into residue-side observations for the two nucleotides in that pair.
Residue Side Choose whether the residue-side y parameter is collected from both nucleotides in each base pair, or only from nt1 or nt2.
